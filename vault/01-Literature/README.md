# Literature & Bibliography

Central repository for sources, citations, and reference materials.

## Structure
- **Papers**: Academic papers and research documents
- **Books**: Book summaries and key insights
- **Web**: Articles, blog posts, and web resources
- **Sources.bib**: BibTeX bibliography file

## Citation Management
Use the [[Citation Plugin]] for managing and citing sources. Access citations via:
```
@citekey
```

For detailed citation info, check the metadata frontmatter in each source file.
