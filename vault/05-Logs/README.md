# Logs & Journals

Daily notes and research journals to track your learning journey.

## Components
- **Daily Notes**: One note per day for captures and reflections
- **Research Journal**: Longer-form reflections on topics
- **Progress Tracking**: Overview of what you've learned

## Features
- Automatic date tagging
- Searchable by date
- Links to referenced sources and notes
- Review previous entries to track growth

## Templates
- [[Daily-Notes-Template]]: Standard daily capture format

## Calendar Integration
Use the Calendar plugin to navigate daily notes by date.
