# Permanent Notes

Well-formed, atomic notes that form the core of your zettelkasten. Each note should:

## Characteristics
- **Atomic**: One idea per note
- **Self-contained**: Understandable without reading others
- **Linked**: Connected to related notes via backlinks
- **Sourced**: Include citations where applicable
- **Unique ID**: Use timestamp-based IDs (e.g., 202406231430)

## Naming Convention
```
[ID] - [Descriptive Title]
Example: 202406231430 - Zettelkasten methodology
```
